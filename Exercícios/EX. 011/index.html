Tags

  • Tag: <audio>.

Parâmetros

  • controls: exibe controles como 'play', 'pause', 'skip' e 'volume'

  • preload: Use esse parâmetro antes de colocar os seguintes:
    • autoplay: site baixa o áudio automaticamente
    • metadata: site baixa apenas algumas informações
    • none: site começa a baixar quando você clicar no 'play'
    • loop: após a finalização do áudio, o site coloca para tocar novamente automaticamente
