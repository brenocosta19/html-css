Código Fonte

O comando document.getElementById('test') é escrito em JavaScript. A tag <code> faz os caracteres do conteúdo selecionado ter a mesma largura.

Para manter a formatação como está no código, basta usar a tag <pre>. Exemplo, se você for copiar um trecho de algum código: 

        
num = int(input('Digite um número'))
if num % 2 = 0:
    print (f 'O número {num} é PAR')
else
    print (f 'O número {num} é IMPAR')
