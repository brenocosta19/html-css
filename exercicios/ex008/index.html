Principais formatações:

Negrito / Destaque

Para colocar um termo em negrito, é possível usando a tag <b>. (não semântica)

Outra forma é através da tag <strong>, dessa forma (semântica)


Itálico / Ênfase

Usando a tag <i>: Itálico (não semântica)

Usando semântica: Ênfase (semântica) = CTRL + SHIFT + P ⇥ ENTER ⇥ EM


Texto Marcado

Dá pra deixar um texto marcado usando a tag <mark>, um exemplo: Hello World !

É possível mudar a cor do texto marcado, mas apenas usando CSS junto da tag STYLE.

Os "marks" são individuais, ou seja, eles não herdam características dos outros, como foi visto nos exemplos acima usando cores diferentes. Se tiver muitos "marks", é necessário colocar um código na parte "Head".

Texto Grande e Pequeno

Para textos GRANDES se usa a tag <big>, e para textos pequenos se usa a tag <small>


Texto Deletado

Textos deletados, como quando erramos na prova, tipo assim: sombrancelha


Texto Inserido e Sublinhado

Texto inserido é para dar ênfase e indicar que ele foi adicionado depois (semântica)

Texto sublinhado não é semântico, e se usa a tag <u>


Texto Sobrescrito e Subscrito

Texto Sobrescrito, exemplo: x20+3

Texto Subscrito, exmplo: H20
